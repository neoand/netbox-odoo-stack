# 🎮 Sistema de Gamificação - NetBox Learning

> **"Aprenda NetBox de forma divertida: Ganhe pontos, conquiste badges e suba no ranking!"**

---

## 🎯 **Conceito**

Sistema completo de gamificação para motivação no aprendizado:
- ✅ **Points System** - Ganhe pontos por atividades
- ✅ **Badges & Achievements** - Conquistas especiais
- ✅ **Leaderboards** - Ranking mensal e global
- ✅ **Challenges** - Desafios semanais e mensais
- ✅ **Rewards** - Prêmios e reconhecimento
- ✅ **Progress Tracking** - Acompanhe seu progresso
- ✅ **Team Competition** - Competição entre equipes
- ✅ **Certificates** - Certificados gamificados

---

## 🏆 **Sistema de Pontos**

### **Earn Points (Ganhar Pontos)**

| Atividade | Pontos | Descrição |
|-----------|--------|-----------|
| **Tutorial Completed** | 100 pts | Completar vídeo tutorial |
| **Quiz Passed** | 50 pts | Aprovar quiz (≥80%) |
| **Lab Exercise** | 150 pts | Finalizar exercício prático |
| **Forum Post** | 25 pts | Post útil no fórum |
| **Code Contribution** | 200 pts | Contribuir com código |
| **Bug Report** | 75 pts | Reportar bug validado |
| **Daily Login** | 10 pts | Login diário (máx 1x/dia) |
| **Weekly Goal** | 500 pts | Completar meta semanal |
| **Monthly Goal** | 2000 pts | Completar meta mensal |
| **Refer Friend** | 300 pts | Indicar amigo que completa curso |
| **Help Others** | 40 pts | Responder dúvidas no Discord |

### **Bonus Multipliers**

| Condição | Multiplicador |
|----------|---------------|
| Streak 7 dias | 1.5x |
| Streak 30 dias | 2.0x |
| Streak 100 dias | 3.0x |
| Early Bird (antes 8h) | 1.2x |
| Night Owl (após 22h) | 1.1x |
| Weekend | 1.3x |
| Perfect Score (100%) | 1.5x |
| First Time | 2.0x |
