# Histórias & Casos Reais

Esta seção foi criada para **conquistar** o coração dos desenvolvedores que nunca ouviram falar do NetBox.

## 📚 Conteúdo

### 1. **[Histórias de Dor](./dores-reais.md)**
Histórias reais de problemas que o NetBox resolve:
- Conflito de IP que parou a produção
- Técnico em campo sem dados
- Custo invisível da falta de integração
- Problema da visibilidade

**Por que ler:** Identifique-se com os problemas e veja como o NetBox seria a solução.

---

### 2. **[PWAs para Time de Campo](./pwas-campo.md)**
Como criar Progressive Web Apps que seus técnicos vão amar usar:
- Scanner de equipamentos
- Checklist de manutenção
- Alertas e notificações
- Mapa de racks
- Sincronização offline

**Por que ler:** Ideias práticas para criar apps que facilitam a vida do time de campo.
