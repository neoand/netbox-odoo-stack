## 🏆 **Como Contribuir**

### **1. Code Contributions**

```bash
# Fork repository
git clone https://github.com/YOUR_USERNAME/wazuh.git
cd wazuh

# Create feature branch
git checkout -b feature/your-feature

# Make changes
# ... write code ...

# Run tests
make test

# Submit PR
git push origin feature/your-feature
# Create PR on GitHub
```

**Areas needing contributions:**
- Decoders (parse custom logs)
- Rules (new detection rules)
- Integrations (third-party tools)
- Documentation (guides, tutorials)
- Plugins (extensions, apps)

### **2. Documentation**
