### **Questões de Exemplo - Bronze:**

**1. No NetBox, qual campo é obrigatório ao criar um device?**
a) Asset tag
b) Serial number
c) Device type
d) Position

**Resposta:** c) Device type

---

**2. Para listar todos os devices via API REST do NetBox, qual endpoint?**
a) GET /api/dcim/sites/
b) GET /api/dcim/devices/
c) GET /api/ipam/ip-addresses/
d) POST /api/dcim/devices/

**Resposta:** b) GET /api/dcim/devices/

---

**3. No Odoo, qual módulo gerencia estoque?**
a) sale
b) purchase
c) stock
d) account

**Resposta:** c) stock
