- Docker Compose 2.0+
- 4GB RAM minimum
- 50GB disk space

### Installation

```bash
# Clone repository
git clone https://github.com/neo-stack/neo_netbox_odoo_stack.git
cd neo_netbox_odoo_stack/platform/certification

# Start all services
docker-compose up -d

# Initialize database
docker-compose exec api python -m scripts.init_db

# Load sample data
docker-compose exec api python -m scripts.load_sample_data

# Access platform
open http://localhost:8080          # Student Portal
open http://localhost:8081          # Admin Portal
```

### Default Credentials

| Portal | URL | Username | Password |
|--------|-----|----------|----------|
| **Student** | http://localhost:8080 | student | student123 |
| **Admin** | http://localhost:8081 | admin | admin123 |

## üì° API Reference

### Authentication

```bash
# Login
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "student", "password": "student123"}'

# Response
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer",
  "expires_in": 3600
}
```

### Exam Management

```bash
# Get available exams
curl -X GET http://localhost:8000/api/exams \
  -H "Authorization: Bearer {token}"

# Start exam
curl -X POST http://localhost:8000/api/exams/{exam_id}/start \
  -H "Authorization: Bearer {token}"

# Submit answer
curl -X POST http://localhost:8000/api/exams/{exam_id}/questions/{question_id} \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"answer": "A"}'
```

### Results

```bash
# Get results
curl -X GET http://localhost:8000/api/exams/{exam_id}/results \
  -H "Authorization: Bearer {token}"

# Response
{
  "exam_id": "exam-001",
  "score": 85,
  "passed": true,
  "answers": [...],
  "certificate_url": "https://..."
}
```

### Certificate Verification

```bash
# Verify certificate
curl -X GET http://localhost:8000/api/certificates/verify/{certificate_id}

# Response
{
  "valid": true,
  "student_name": "John Doe",
  "certification_level": "Professional",
  "issue_date": "2025-12-06",
  "certificate_url": "https://..."
}
```
